Семантическая сегментация спутниковых снимков (LoveDA)
Данный репозиторий содержит исходный код проекта, выполненного в рамках производственной (проектно-технологической) практики.

Тема: Применение искусственных нейронных сетей для решения прикладных задач семантической сегментации в компьютерном зрении. Выделение зданий и дорог на спутниковых снимках (Вариант 30).

Демонстрационный веб-модуль на Hugging Face Spaces:
https://huggingface.co/spaces/ROIT19/LLM_30var

Описание проекта
Целью проекта является автоматизация извлечения геопространственной информации (зданий, дорог, лесных массивов) из спутниковых снимков плотной городской застройки.
В ходе работы была произведена подготовка датасета LoveDA, а также обучены и протестированы 5 различных архитектур нейронных сетей (энкодер ResNet-34):

U-Net

LinkNet

PSPNet

DeepLabV3

FPN (Feature Pyramid Network) — показала наилучший результат.

Результаты экспериментального сравнения
Наилучшую эффективность продемонстрировала архитектура FPN:

Pixel Accuracy: 93.23%

Loss (Cross-Entropy): 0.1603

Модель успешно справляется с выделением сложной геометрии зданий и непрерывным сегментированием автомагистралей, включая распознавание объектов с сильным варьированием масштабов. Именно эта модель интегрирована в итоговое веб-приложение.

Структура репозитория
app.py — исходный код веб-приложения на базе фреймворка Gradio для визуализации инференса.

requirements.txt — список зависимостей и библиотек, необходимых для работы проекта.

(Опционально) Jupyter Notebook с конвейером обучения моделей.

Запуск проекта локально
Для запуска веб-интерфейса на локальной машине выполните следующие шаги:

Клонируйте репозиторий:

Bash
git clone https://github.com/vetrovbvt2352/Var30.git

Установите необходимые зависимости:

Bash
pip install -r requirements.txt
Запустите приложение:

Bash
python app.py
После запуска интерфейс будет доступен в браузере по локальному адресу http://127.0.0.1:7860.
